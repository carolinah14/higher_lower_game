# Higher_Lower_Game (Python)

A Python console-based “Higher or Lower” game where the user compares two Instagram accounts and guesses which one has more followers, using data from external modules. The game continues as long as the user guesses correctly, keeping track of the score and carrying forward the winning account for the next round.

# ▶️ How to Run

1. Make sure you have Python installed on your computer.

2. Download or clone this repository.

3. Ensure all files are in the same folder:

    higher_or_lower.py

    art.py

    game_data.py

4. Open a terminal (or your preferred Python IDE).

5. Navigate to the folder containing the files.

6. Run the script using:

   python higher_or_lower.py

7. Follow the on-screen prompts to play the game.
